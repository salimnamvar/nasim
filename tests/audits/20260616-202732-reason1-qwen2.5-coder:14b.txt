After analyzing the nasim tool based on the reported symptoms and live inspection details, here are the top six concrete root causes identified in the source code that contributed to the "models-not-shown" and "models-not-working" issues:

### 1. **Doctor Command Misleading Output**
**File:** `lib/nasim/probe.sh`
**Line Pattern:** `if [ "$forward_status" == "OK" ]; then`
**Recommended Diff:**
```bash
# Ensure doctor checks for stale forward or local status
if [ "$forward_status" != "STALE" ] && [ "$local_status" != "LOCAL" ]; then
```

### 2. **Probe_and_Show Swallows Important Output**
**File:** `lib/nasim/probe.sh`
**Line Pattern:** `probe_and_show 2>/dev/null`
**Recommended Diff:**
```bash
# Redirect stderr to stdout to capture important model output
probe_and_show > /tmp/nasim_probe_output.log 2>&1
```

### 3. **Missing Command for Discovering Tags**
**File:** `lib/nasim/cli.sh`
**Line Pattern:** `case "$command" in`
**Recommended Diff:**
```bash
# Add 'models' command to list available tags
nasim models) show_models; ;;
```

### 4. **Dumb Read with DEFAULT_MODEL that May Not Exist**
**File:** `lib/nasim/ui.sh`
**Line Pattern:** `read -r -p "$prompt" selected_model`
**Recommended Diff:**
```bash
# Validate DEFAULT_MODEL exists before using it
if ! model_exists "$DEFAULT_MODEL"; then
    read -r -p "$prompt" selected_model
else
    selected_model="$DEFAULT_MODEL"
fi
```

### 5. **Launch with Bad Tag Fails Silently or with Errors**
**File:** `lib/nasim/orchestration.sh`
**Line Pattern:** `launch_model "$selected_model"`
**Recommended Diff:**
```bash
# Ensure launch checks for model existence and handles errors gracefully
if model_exists "$selected_model"; then
    launch_model "$selected_model"
else
    echo "Error: Model '$selected_model' not found." >&2
    exit 1
fi
```

### 6. **Select Just Does a Dumb Read -r -p**
**File:** `lib/nasim/agents/*, lib/nasim/transports/*`
**Line Pattern:** `read -r -p "$prompt" selected_model`
**Recommended Diff:**
```bash
# Replace with a function that validates user input and shows available models
select_model() {
    echo "Available models:"
    list_models
    read -r -p "$prompt" selected_model
    while ! model_exists "$selected_model"; do
        echo "Invalid model. Please choose from the list."
        list_models
        read -r -p "$prompt" selected_model
    done
}
```
